# upload-oss-sample

## 功能说明

配置 ossClient 信息后即可拖拽或选择文件并上传到 OSS

![upload](https://img.alicdn.com/imgextra/i3/O1CN01ieyFRp1Qhi8PqOXoT_!!6000000002008-2-tps-2696-1804.png)

## 运行
```bash
# 开发
$ npm run watch

# 编译
$ npm run compile

# 运行
$ sumi dev
```
